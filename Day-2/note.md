# Đường dẫn tương đối
Là đường dẫn tính từ vị trí file hiện tại, không cần ghi đầy đủ từ gốc.
Ví dụ:
Giả sử đang ở file index.html

<img src="images/logo.png">
## Đường dẫn tuyệt đối
Là đường dẫn đầy đủ từ thư mục gốc, không phụ thuộc vị trí file hiện tại.

Ví dụ (trên máy tính):

C:/Users/Admin/Desktop/web/images/logo.png


Ví dụ (trên web):

https://example.com/images/logo.png

### Phân biệt các loại đường dẫn 

1. /duong dan --> Tuyệt đối

2. ../duong dan --> Tương đối

3. duong-dan --> Tương đối

4. ./duong-dan --> Tuyệt đối

5. .//domain.com/duong-dan --> Tuyệt đối

6. https://domain.com/duong-dan --> Tuyệt đối
